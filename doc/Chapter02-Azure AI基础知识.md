# 2.1 什么是人工智能（Artificial Intelligence）

人工智能（Artificial Intelligence，简称 **AI**）是指让计算机模拟人类智能行为的技术，使其能够完成通常需要人类智慧才能完成的任务，例如学习、推理、识别、理解语言和解决问题。

AI 并不是一种单一技术，而是由多种技术共同组成，包括：

* 机器学习（Machine Learning）
* 深度学习（Deep Learning）
* 自然语言处理（Natural Language Processing，NLP）
* 计算机视觉（Computer Vision）
* 语音智能（Speech AI）
* 生成式 AI（Generative AI）

这些技术共同构成了现代 AI 的核心能力。

---

# 2.2 AI、机器学习、深度学习与生成式 AI

很多初学者容易混淆这些概念，它们之间的关系如下：

```text
人工智能（AI）
│
├── 机器学习（Machine Learning）
│      │
│      └── 深度学习（Deep Learning）
│
└── 生成式 AI（Generative AI）
```

### 人工智能（AI）

广义概念，目标是让计算机具备类似人类的智能。

### 机器学习（Machine Learning）

机器学习是 AI 的一种实现方式，通过大量数据训练模型，让计算机能够自动发现规律并进行预测，而无需为每种情况编写规则。

例如：

* 垃圾邮件识别
* 商品推荐
* 信用风险评估

### 深度学习（Deep Learning）

深度学习是机器学习的一个分支，通过多层神经网络学习复杂特征，特别适合图像、语音和自然语言处理等任务。

例如：

* 人脸识别
* 图像分类
* 语音识别

### 生成式 AI（Generative AI）

生成式 AI 不仅能够分析数据，还能够根据已有知识生成新的内容，例如：

* 撰写文章
* 生成图片
* 编写代码
* 总结文档
* 对话问答

目前广泛使用的大语言模型（LLM）就是生成式 AI 的典型代表。

# 2.3 常见 AI 工作负载（AI Workloads）

AI 工作负载是指 AI 系统需要完成的任务类型。

| 工作负载                        | 示例              |
| --------------------------- | --------------- |
| 计算机视觉（Computer Vision）      | 图像识别、目标检测、人脸检测  |
| 自然语言处理（NLP）                 | 情感分析、文本分类、关键词提取 |
| 语音 AI（Speech AI）            | 语音识别、语音合成、实时翻译  |
| 文档智能（Document Intelligence） | 发票识别、表单解析、OCR   |
| 知识挖掘（Knowledge Mining）      | 企业文档搜索、智能问答     |
| 生成式 AI（Generative AI）       | 文本生成、代码生成、图片生成  |

在 AI-901 考试中，经常会给出一个业务场景，要求选择最适合的 AI 工作负载，因此理解各种工作负载的特点非常重要。

# 2.4 Azure AI 的核心组成

Azure 提供了完整的 AI 平台，主要包括以下几个部分：

| 服务                     | 作用                   |
| ---------------------- | -------------------- |
| Azure AI Foundry       | 构建、管理和部署生成式 AI 应用    |
| Azure AI Services      | 提供视觉、语音、语言等预构建 AI 能力 |
| Azure Machine Learning | 模型训练、评估和部署           |
| Azure OpenAI Service   | 调用大型语言模型（LLM）能力      |

这些服务相互协作，帮助开发者快速构建 AI 应用。

# 2.5 AI 在企业中的应用

Azure AI 已广泛应用于各行各业，例如：

| 行业 | 应用场景             |
| -- | ---------------- |
| 零售 | 商品推荐、客服机器人、需求预测  |
| 金融 | 风险评估、欺诈检测、智能客服   |
| 医疗 | 医学影像分析、病历整理、辅助诊断 |
| 制造 | 缺陷检测、预测性维护       |
| 教育 | 智能学习助手、自动评分      |
| 办公 | 文档总结、会议纪要、知识检索   |

企业通常会根据实际需求选择不同的 Azure AI 服务，而不是自行训练复杂模型。

# 2.6 为什么选择 Azure AI？

Azure AI 具有以下优势：

* 提供丰富的预构建 AI 服务。
* 支持生成式 AI 与大型语言模型（LLM）。
* 可与 Azure 云服务无缝集成。
* 提供企业级安全、身份验证与权限管理。
* 支持 SDK、REST API 等多种开发方式。
* 可快速部署 AI 应用，降低开发门槛。

对于企业而言，Azure AI 能够帮助缩短开发周期，提高 AI 应用落地效率。
