---

# 1.1 什么是 AI-901？

**AI-901：Microsoft Azure AI Fundamentals** 是微软推出的 Azure AI 基础认证，主要考查考生对人工智能（AI）基础概念、生成式 AI、Responsible AI，以及 Azure AI 服务的理解与应用能力。

AI-901 面向 AI 初学者，不要求掌握复杂的机器学习算法，也无需具备丰富的开发经验。学习重点在于理解 Azure AI 提供的各项服务，并能够根据业务需求选择合适的 AI 解决方案。

对于希望进入 Azure AI 领域的学习者来说，AI-901 是一个良好的起点。

---

# 1.2 AI-901 适合哪些人？

AI-901 适合以下人群：

* AI 初学者
* 云计算初学者
* 软件开发工程师
* 测试工程师
* 数据分析师
* 产品经理
* IT 运维人员
* 高校学生
* 希望了解 Azure AI 的技术人员

即使没有人工智能相关经验，也可以通过本认证建立完整的 Azure AI 知识体系。

# 1.3 考试信息

| 项目   | 内容                                      |
| ---- | --------------------------------------- |
| 考试名称 | AI-901: Microsoft Azure AI Fundamentals |

# 1.4 AI-901 学习内容

本书将围绕以下几个模块展开：

| 模块                | 内容                                           |
| ----------------- | -------------------------------------------- |
| AI 基础             | AI、机器学习、生成式 AI、Responsible AI                |
| Azure AI Foundry  | 模型、Prompt、Agent、Knowledge、部署                 |
| Azure AI Services | Vision、Speech、Language、Document Intelligence |
| 开发实践              | Azure Portal、Python SDK、REST API             |
| 考试冲刺              | 高频考点、易错点、模拟试题                                |

学习过程中，建议遵循"先理解概念，再学习服务，最后实践验证"的思路。

---

# 1.5 AI-901 能力要求

完成 AI-901 学习后，应具备以下能力：

* 理解 AI 与生成式 AI 的基本概念。
* 理解 Responsible AI 的核心原则。
* 熟悉 Azure AI Foundry 的基本功能。
* 了解 Azure AI Services 的主要能力。
* 能根据业务需求选择合适的 Azure AI 服务。
* 了解 Azure AI 的基本开发方式（SDK、REST API）。

AI-901 更关注**理解与应用**，而不是底层算法实现。

# 1.7 学习建议

为了提高学习效率，建议采用以下方法：

1. **理解概念，而不是死记硬背。**
2. **结合 Azure Portal 动手实践。**
3. **总结各项 Azure AI 服务的适用场景。**
4. **整理易混概念，对比记忆。**
5. **每章完成后及时练习，巩固知识。**

持续学习和实践，比一次性大量阅读更有效。
