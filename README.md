# Azure AI-901 Study Guide（中文版）

> 📚 一本面向 **Microsoft Azure AI-901: Azure AI Fundamentals** 认证考试的中文学习手册。

本项目旨在帮助学习者系统掌握 Azure AI 基础知识，并顺利通过 AI-901 认证考试。同时，本手册结合 Azure AI Foundry、Azure AI Services、Python SDK、REST API 等内容，帮助读者从"了解概念"过渡到"能够实践"。

---

# 本书特点

* 📖 根据最新 AI-901 考试大纲编写
* 🌐 全中文讲解，适合中文学习者
