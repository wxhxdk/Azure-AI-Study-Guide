# Azure AI Study Guide

> 一个面向开发者的 Azure AI 学习手册。
> 从 Azure 基础开始，逐步学习 Azure AI 服务、开发实践以及认证（AI-900 / AI-102）。

---

## 📖 项目简介

本项目旨在构建一套持续维护、循序渐进的 Azure AI 学习资料。

不同于单纯的考试笔记，本项目更加注重：

* 建立完整的 Azure AI 知识体系
* 理解技术原理，而不仅仅是 API 的使用
* 结合 Microsoft Learn 与官方文档学习
* 配套实验、示例代码和实践项目
* 持续更新，跟进 Azure AI 最新发展

最终目标是帮助开发者：

* 理解 Azure AI 的整体架构
* 通过 Azure AI 相关认证
* 能够独立开发 Azure AI 应用
